## 🚀 Featured Work

### [llm-eval-framework](https://github.com/milecfranca/llm-eval-framework)
An automated testing and quality assurance framework designed to evaluate Generative AI text outputs for production readiness. It systematically assesses LLM generations against source contexts using automated judging metrics to compute factual consistency, isolate model hallucinations, and enforce strict domain tone alignment.

**Key Features:**
- Factual consistency scoring
- Hallucination detection
- Tone alignment verification
- Scalable evaluation pipelines

**Tech Stack:** `Python` `LLM APIs` `Data Curation` `Automated Evaluation`
